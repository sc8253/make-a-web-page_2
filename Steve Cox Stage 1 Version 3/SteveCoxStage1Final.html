Important Concepts

Getting Started with HTML - The Basics of the Web and HTML

Elements of the Web

The Web is made up of HTML, URL's, HTTP, and various Web applications.

HTML

HTML stands for Hypertext Markup Language. This is the main type of document on the world wide web. Hyper links HTML is the glue holding the 30 billion pages on the web together. HTML documents contain text content which describes "what you see" and markup which describes "how it looks".

File Types

Plain Text, HTML, images, video, and music are all file types found on the web.

Components of the Web

User (cpu and browser), the Internet, HTTP, and servers are all components on the web. Hyper Text Transfer Protocol is a main protocol on the web and allows communication between users, the browswer and servers over the Inernet. The browser is a program that runs on your computer and displays files from web servers.

HTML Basics

Text content - the what you see
Markup - what it looks like
References to other documents, images, or videos
Links - refereces to other pages or documents

HTML Markup

The block paragraph form of code is an "element"
HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them.

Creating a Structured Document with HTML

Developer Tools

Develper tools allow you to see the HTML elements of a web page and edit from a web browser. Google Chrome allows in browser editing of web pages through Dev Tools.

Webpages, Documents, and Structure

Three main languages:
HTML - structure of the page
CSS - style of the page
Java - interavtive changing elements

Tree Stucture of HTML

HTML elements work in a tree like structure. HTML classifies content and structures the page.
Document Object Model - is a cross-platform and language-independent convention for representing and interacting with objects in HTML, XHTML, and XML documents. The nodes of every document are organized in a tree structure, called the DOM tree.

Code Editing Tools

Programmers use text editing tools such as Codepen, Scratchpad, and Sublime Text to write code. When viewing code in a text editor, the indentations on the left hand side correspond to the layers of nesting in a particular element or box on the page.

Lesson 3: Adding CSS Style to HTML Structure

Repetition

Repetition in coding will often lead to rework due to errors. Consistency in form leads to decreased errors and the ability to make changes easily to style.

Cascading Style Sheets

CSS controls and forms the style of the HTML element via class rules. The most specific rule applies to every element.
